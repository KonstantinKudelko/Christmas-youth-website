# Server

Here user can create articles and other content

# API

| action        | request_method | request_url          |
| ------------- |:--------------:| --------------------:|
| articles list | GET            | ${host}/articles     |
| one article   | GET            | ${host}/articles/:id |
| authors list  | GET            | ${host}/authors      |
| tags list     | GET            | ${host}/tags         |
